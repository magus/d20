# d20
roll a d20

# setup
```
yarn
```

# dev
```
yarn dev
```
**Open [http://localhost:3000](http://localhost:3000) to view it in the browser.**

# deploy
```
yarn now
```
**Open [https://d20.now.sh](https://d20.now.sh) to view it in the browser.**

## Clean up old deploys
```
yarn now rm d20 --safe
```
